# TaskFlow

A team task manager where you can create projects, assign tasks, and track progress with your team.

## What it does

- Sign up and log in securely
- Create projects and add team members
- Add tasks with priorities and due dates
- Move tasks across a Kanban board (To Do → In Progress → Done)
- Dashboard shows your tasks, overdue alerts, and progress stats

## Roles

There are two roles — **Admin** and **Member**.

Admins can create projects, manage team members, and edit any task. Members can only see projects they're added to and manage their own tasks.

## Tech used

- Node.js + Express
- NeDB (simple file-based database)
- Vanilla JavaScript 
- JWT for authentication
